# Wild Hog Canoe Race Timekeeper

Local startup:

1. `npm install`
2. `npm run dev`
3. Open `http://127.0.0.1:8000/`

The standalone local server uses plain Node.js, serves the operator app at `/`, the coordinator board at `/coordinator`, and reuses the existing API handlers under `/api/*` without requiring `vercel dev`, `vercel login`, or Vercel CLI authentication.
